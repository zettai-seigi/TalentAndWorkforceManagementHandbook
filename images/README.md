## Visual Specification Format

Each visual specification includes:

1. **Visual ID**: Unique identifier (e.g., 07-Figure-7.1)
2. **Type**: Visual format (Flowchart, Matrix, Infographic, Dashboard, etc.)
3. **Title**: Descriptive title for the visual
4. **Chapter Reference**: Specific chapter and section location
5. **Purpose Description**: Clear explanation of visual's objective
6. **Content Description**: Detailed breakdown of structure, flow, and key elements
7. **Specifications**: Technical details including:
   - Dimensions (pixels)
   - Color palette (consistent hex codes)
   - Fonts and typography
   - Layout specifications
   - Recommended tools (Figma, Lucidchart, PowerPoint, etc.)
8. **Caption**: Publication-ready caption text
9. **Position in Chapter**: Exact placement within chapter structure
10. **Notes**: Additional guidance, templates, or considerations

## Accessibility Standards

All visuals must meet:

### Color
- ✓ Contrast ratio ≥ 4.5:1 for normal text
- ✓ Contrast ratio ≥ 3:1 for large text
- ✓ Color is not the only means of conveying information

### Text
- ✓ Minimum font size: 11pt for body text
- ✓ Clear, sans-serif fonts (Arial, Helvetica, Calibri)
- ✓ Alt text descriptions for all visuals

### Design
- ✓ Patterns/shapes in addition to colors
- ✓ Clear labels and legends
- ✓ High-resolution images (minimum 150 DPI for print)
