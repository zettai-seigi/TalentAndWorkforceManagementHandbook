# Bab 14: Model Kematangan dan Penilaian

## Tujuan Pembelajaran

- Memahami model kematangan 5-tingkat
- Melakukan penilaian kematangan organisasi
- Mengidentifikasi area peningkatan
- Mengembangkan peta jalan peningkatan

---

## Model Kematangan 5-Tingkat

### Gambaran Tingkat

| Tingkat | Nama | Karakteristik |
|---------|------|---------------|
| **1** | Initial | Ad-hoc, reaktif, tidak konsisten |
| **2** | Managed | Proses dasar ada, beberapa dokumentasi |
| **3** | Defined | Proses standar, diikuti konsisten |
| **4** | Measured | Metrik digunakan untuk pengambilan keputusan |
| **5** | Optimized | Peningkatan berkelanjutan, inovasi |

---

## Dimensi Penilaian

### 8 Dimensi Kapabilitas

| Dimensi | Deskripsi |
|---------|-----------|
| **1. Planning** | Perencanaan tenaga kerja |
| **2. Acquisition** | Akuisisi bakat |
| **3. Development** | Pembelajaran & pengembangan |
| **4. Performance** | Manajemen kinerja |
| **5. Engagement** | Keterlibatan karyawan |
| **6. Retention** | Retensi bakat |
| **7. Technology** | Teknologi HR |
| **8. Governance** | Tata kelola |

---

## Proses Penilaian

### Langkah Penilaian

1. **Persiapan** - Bentuk tim, kumpulkan dokumen
2. **Pengumpulan Data** - Wawancara, review dokumen
3. **Scoring** - Nilai per dimensi
4. **Analisis** - Identifikasi kekuatan & gap
5. **Pelaporan** - Dokumentasi temuan
6. **Planning** - Buat peta jalan peningkatan

### Template Peta Jalan

| **1** | 0-6 bulan | Level 2 | Foundational |

## Ringkasan

Model kematangan memungkinkan:
- Penilaian objektif kondisi saat ini
- Identifikasi prioritas peningkatan
- Perencanaan peta jalan yang realistis
- Tracking progress sepanjang waktu
