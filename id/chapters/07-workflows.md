# Bab 7: Alur Kerja dan Proses

## Tujuan Pembelajaran

- Merancang alur kerja tenaga kerja end-to-end
- Mengimplementasikan proses perekrutan yang efisien
- Mengelola proses tinjauan kinerja
- Mengoptimalkan proses offboarding

---

## Alur Kerja Utama

### 1. Alur Kerja Perekrutan

```
Requisition → Approval → Posting → Sourcing → Screening → Interview → Offer → Onboarding
```

### 2. Alur Kerja Tinjauan Kinerja

```
Goal Setting → Mid-Year Review → Year-End Review → Calibration → Feedback
```

### 3. Alur Kerja Pelatihan

```
Needs Assessment → Planning → Delivery → Evaluation → Follow-up
```

### 4. Alur Kerja Offboarding

```
Resignation → Exit Interview → Knowledge Transfer → Asset Return → Access Revocation → Alumni
```

---

## Proses Perekrutan Detail

### Tahapan Proses

| Tahap | Aktivitas | Durasi Target | Output |
|-------|-----------|---------------|--------|
| **1. Requisition** | Submit & approval | 3 hari | Req approved |
| **2. Posting** | Publish ke saluran | 1 hari | Job posted |
| **3. Sourcing** | Aktif mencari kandidat | Ongoing | Pipeline kandidat |
| **4. Screening** | Review CV | 5 hari | Shortlist |
| **5. Interview** | 2-3 ronde | 10 hari | Final candidate |
| **6. Offer** | Nego & offer letter | 5 hari | Offer accepted |
| **7. Onboarding** | Persiapan mulai | 5 hari | Siap mulai |

---

## Proses Manajemen Kinerja

### Siklus Tahunan

| Kuartal | Aktivitas |
|---------|-----------|
| **Q1** | Goal setting untuk tahun baru |
| **Q2** | Mid-year check-in |
| **Q3** | Feedback dan coaching |
| **Q4** | Year-end review, calibration |

---

## Proses Offboarding

### Checklist Offboarding

- [ ] Pemberitahuan ke HR dan stakeholder
- [ ] Exit interview
- [ ] Knowledge transfer plan
- [ ] Project handover
- [ ] Pengembalian aset
- [ ] Pencabutan akses sistem
- [ ] Final payment dan dokumentasi
- [ ] Alumni program invitation

---

## Ringkasan

Alur kerja yang terdefinisi dengan baik memastikan:
- Konsistensi dalam eksekusi
- Efisiensi waktu dan biaya
- Pengalaman karyawan yang positif
- Kepatuhan terhadap kebijakan
