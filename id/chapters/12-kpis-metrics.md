# Bab 12: Indikator Kinerja Utama dan Metrik

## Tujuan Pembelajaran

- Mengidentifikasi KPI tenaga kerja yang relevan
- Merancang sistem pengukuran yang komprehensif
- Menetapkan target dan benchmark
- Menggunakan metrik untuk pengambilan keputusan

---

## Kerangka KPI

### Kategori KPI Tenaga Kerja

| Kategori | KPI | Target |
|----------|-----|--------|
| **Akuisisi** | Time to Fill | < 45 hari |
| | Cost per Hire | Bervariasi |
| | Quality of Hire | > 3.5/5 |
| **Retensi** | Turnover Rate | < 15% |
| | Retention Rate | > 85% |
| **Pengembangan** | Training Hours | > 40 jam/tahun |
| | Competency Coverage | > 90% |
| **Kinerja** | Performance Rating Avg | > 3.0/5 |
| | Goal Achievement | > 80% |
| **Engagement** | Employee Engagement | > 75% |
| | eNPS | > 30 |

---

## KPI Detail

### 1. Time to Fill

| Aspek | Detail |
|-------|--------|
| **Definisi** | Hari dari requisition approved sampai offer accepted |
| **Rumus** | Offer Accept Date - Req Approval Date |
| **Target** | < 45 hari |
| **Frekuensi** | Bulanan |

### 2. Employee Turnover Rate

| Aspek | Detail |
|-------|--------|
| **Definisi** | Persentase karyawan yang keluar dalam periode |
| **Rumus** | (Terminations / Avg Headcount) × 100% |
| **Target** | < 15% per tahun |
| **Breakdown** | Voluntary vs Involuntary |

### 3. Training Hours per Employee

| Aspek | Detail |
|-------|--------|
| **Definisi** | Rata-rata jam pelatihan per karyawan |
| **Rumus** | Total Training Hours / Headcount |
| **Target** | > 40 jam/tahun |
| **Include** | Formal + informal learning |

## Ringkasan

KPI yang efektif harus:
- Selaras dengan strategi bisnis
- Terukur dan actionable
- Di-review secara berkala
- Digunakan untuk pengambilan keputusan
